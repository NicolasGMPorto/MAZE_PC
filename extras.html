Criação

Este projeto foi um protótipo de um trabalho de EAMT, tendo a professora Talita como responsável, o intuito do site era ser um site com algumas informações sobre o jogo que iríamos fazer futuramente, além de contar segredos importante para a Lore da história principal, porém, ele ficou tão grande que praticamente virou um jogo próprio, infelizmente, teve uma mudança de planos e esse projeto foi abandonado, mas como ele ficou bem feito e está praticamente completo, decidimos compartilhar com vocês, aproveitem!

Códigos e Avisos

O que fazer para ativar o jogo ?

Primeiro Passo: Aperte algumas vezes no título que está no Menu Principal (dentro da zona preta), isso vai liberar uma nova opção secreta

Segundo Passo: Após entrar nessa zona secreta, aperte no cadeado que tem no canto inferior direito da página para abrir o input de códigos

Terceiro Passo: Nesse espaço, você pode digitar alguns códigos para acessar alguns easter eggs, lembre-se, como a criação do site foi cortada no meio, então alguns jogos podem apresentar bugs ou terem uma estética não terminada, ou também podem não ter responsabilidade para telas maiores, então caso ocorra esse tipo de coisa, saiba que é natural

Quarto Passo: Use esses códigos para acessar os minigames:

"Sim"
"disk_181"
"Pega_Lixo"
"Nightmare"

Ao Final de cada minigame, terá 2 elementos
O código, se colocado no input de códigos na aba secreta, desbloqueará o modo difícil do minigame que você jogou anteriormente
O número, se combinado com os números dos outros 4 minigames, te levará para o boss final, tome cuidado
Para ativar o bossfinal, coloque os números em ordem crescente
